# 🚀 Job Hunt Dashboard

## 📌 Overview

Job Hunt Dashboard is a lightweight web application designed to streamline and optimize the job application process for software developers. It provides tools to manage resumes, generate personalized outreach emails, track HR interactions, and maintain daily consistency in job hunting.

The goal is to eliminate chaos and replace it with a structured, repeatable system.

---

## 🎯 Objectives

- Centralize all job-hunting activities in one place
- Reduce manual effort in writing emails and tracking outreach
- Improve consistency and discipline
- Increase response and interview rates
- Serve as a productivity + portfolio project

---

## 🧩 Core Features

### 📄 1. Resume Manager

- Store and view multiple resume versions:
  - Full Stack Resume
  - Backend Resume
  - Systems/Startup Resume

- Quick switching between resumes
- Copy-ready format

---

### 📬 2. Email Generator

- Input fields:
  - HR Name
  - Company Name
  - Role (optional)

- Auto-generates personalized email
- Multiple templates:
  - Cold outreach
  - Follow-up
  - Referral request

- Copy-to-clipboard functionality

---

### 🗂 3. HR Tracker

- Add HR details:
  - Name
  - Company
  - Contact status
  - Notes (optional)

- View all entries in table format
- Track:
  - Contacted / Not Contacted
  - Follow-up pending

- Data stored in browser (localStorage)

---

### 📊 4. Daily Goal Tracker

- Add daily tasks like:
  - “Send 5 emails”
  - “Connect with 5 HRs”

- Mark tasks as completed
- Persist tasks using localStorage
- Helps maintain consistency

---

### 📥 5. Export to Excel (CSV)

- Export HR data into `.csv`
- Can be opened in Excel / Google Sheets
- Enables tracking outside app
